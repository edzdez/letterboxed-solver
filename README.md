# letterboxed-solver-rs
A Letter Boxed solver written in Rust

## To Build
```shell
$ git clone "https://github.com/edzdez/letterboxed-solver-rs.git"
$ cd ./letterboxed-solver-rs
$ cargo build --release
```

## To Run
1. Edit `letterbox.in` with the letters around the square
2. ```shell
   $ cargo run --release
   ```
